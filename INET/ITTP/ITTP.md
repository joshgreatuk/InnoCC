# Notes
The ITTP library is a collection of software to support a http-like protocol integrated with rednet

The core library consists of:
- ITTPLib
- ITTPClient
- ITTPServer
- ITTPRouter
- ITTPDNS

In the future the ITTP library can be secured with INNS (INNO Security)
